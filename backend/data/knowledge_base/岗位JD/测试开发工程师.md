# 测试开发工程师

## 岗位职责

- 负责AI应用系统的测试策略制定和执行
- 开发自动化测试框架和工具，提升测试效率
- 设计和执行功能测试、性能测试、安全测试等
- 参与需求评审和代码审查，从测试角度提出改进建议
- 建立和维护测试环境，管理测试数据
- 分析和跟踪缺陷，推动问题解决
- 输出测试报告和质量分析报告

## 技术要求

- 精通Python编程，具备良好的编码能力
- 熟悉pytest/unittest等测试框架
- 了解AI/ML应用的测试方法和工具
- 熟悉API测试工具（Postman/requests等）
- 了解性能测试工具（Locust/JMeter等）
- 熟悉CI/CD流程和工具（Jenkins/GitHub Actions等）
- 了解Docker和容器化测试环境
- 有LLM应用测试经验者优先

## 常见技能

- Python
- pytest / unittest
- API测试
- 性能测试
- 自动化测试
- CI/CD
- Docker
- Git
- 测试用例设计
- 缺陷管理

## 面试重点

- 测试用例设计方法（等价类、边界值、场景法等）
- 自动化测试框架的设计和实现
- API测试的最佳实践
- 性能测试的方案设计和执行
- AI应用的测试挑战和解决方案
- CI/CD流程的搭建和优化
- 如何保证测试覆盖率
- 缺陷分析和质量度量

## 项目要求

- 有完整的自动化测试框架开发经验
- 有API测试和性能测试经验
- 有CI/CD流程搭建经验
- 能够独立制定测试策略和计划

## 初级岗位能力标准

- 能够使用pytest编写和执行测试用例
- 能够进行基本的API测试
- 能够设计合理的测试用例
- 能够使用Git进行版本管理
- 能够输出清晰的测试报告
