# Python后端开发

## 岗位职责

- 负责公司后端服务的设计、开发和维护
- 使用Python/Go等语言开发高性能、高可用的后端服务
- 设计和实现RESTful API接口，与前端和其他服务进行对接
- 负责数据库设计和优化，包括关系型数据库和NoSQL数据库
- 参与系统架构设计，保证系统的可扩展性和可维护性
- 编写技术文档和单元测试，保证代码质量

## 技术要求

- 精通Python编程，熟悉常用数据结构和算法
- 熟悉Django/Flask/FastAPI等Web框架
- 熟悉MySQL/PostgreSQL等关系型数据库，了解Redis等缓存数据库
- 了解消息队列（RabbitMQ/Kafka）的使用
- 熟悉RESTful API设计规范
- 了解Docker容器化部署
- 熟悉Linux操作系统和常用命令
- 有微服务架构经验者优先

## 常见技能

- Python
- FastAPI / Django / Flask
- MySQL / PostgreSQL
- Redis
- Docker
- Git
- RESTful API
- 消息队列
- Linux
- Nginx

## 面试重点

- Python的高级特性（装饰器、生成器、上下文管理器等）
- Web框架的原理和使用
- 数据库设计和SQL优化
- RESTful API设计原则
- 高并发场景的处理方案
- 缓存策略和使用场景
- Docker容器化部署
- 常见设计模式的应用

## 项目要求

- 有完整的后端项目开发经验
- 熟悉从需求分析到上线部署的全流程
- 有数据库设计和优化经验
- 有API接口设计和开发经验

## 初级岗位能力标准

- 能够使用FastAPI/Django开发后端服务
- 能够设计和实现RESTful API
- 能够进行基本的数据库设计和操作
- 能够使用Docker进行容器化部署
- 能够编写单元测试和文档
