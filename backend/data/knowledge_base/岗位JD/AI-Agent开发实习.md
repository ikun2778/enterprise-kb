# AI Agent开发实习

## 岗位职责

- 参与AI Agent产品的设计与开发，包括对话系统、工具调用、任务编排等核心模块
- 基于LangChain/LlamaIndex等框架构建Agent应用，实现多工具协同调用
- 负责Prompt Engineering，优化Agent的推理能力和输出质量
- 参与RAG系统开发，包括文档解析、向量检索、混合检索等模块
- 协助进行模型评估、数据分析和效果优化

## 技术要求

- 熟悉Python编程，具备良好的编码习惯
- 了解LangChain框架，有Agent/RAG开发经验者优先
- 了解大语言模型（LLM）的基本原理和调用方式
- 熟悉向量数据库（FAISS/Milvus/Chroma）的基本使用
- 了解Prompt Engineering的基本技巧
- 熟悉Git版本管理工具

## 常见技能

- Python
- LangChain
- RAG
- FAISS
- OpenAI API / Moonshot API
- Prompt Engineering
- 向量数据库
- FastAPI
- Git

## 面试重点

- RAG系统的完整流程（文档加载→分块→Embedding→检索→生成）
- Agent的ReAct模式和Tool Calling机制
- 向量检索vs关键词检索的区别和适用场景
- 父子文档检索的原理和优势
- Prompt Engineering的实践经验
- 项目中的技术选型和优化经验

## 项目要求

- 有RAG或Agent相关的个人项目经验
- 能够清晰描述项目架构、技术选型和优化过程
- 有实际的API调用和LLM应用开发经验
- 项目最好包含：向量检索、混合检索、多轮对话等功能

## 初级岗位能力标准

- 能够独立搭建基础的RAG系统
- 能够使用LangChain构建简单的Agent应用
- 能够进行基本的Prompt优化和调试
- 理解Embedding和向量检索的基本原理
- 能够使用FastAPI构建API接口
