# 食物中毒预防与处理

## 一、什么是食物中毒

食物中毒是指摄入了含有生物性、化学性有毒有害物质的食品或把有毒有害物质当作食品摄入后出现的非传染性急性、亚急性疾病。

### 1.1 食物中毒的分类

1. **细菌性食物中毒**：最常见，如沙门氏菌、金黄色葡萄球菌
2. **化学性食物中毒**：农药残留、重金属超标
3. **真菌性食物中毒**：霉变食品、毒蘑菇
4. **动物性食物中毒**：河豚鱼、贝类毒素
5. **植物性食物中毒**：发芽土豆、未煮熟的四季豆

## 二、常见食物中毒症状

### 2.1 胃肠道症状

- 恶心、呕吐
- 腹痛、腹泻
- 发热、寒战

### 2.2 神经系统症状

- 头晕、头痛
- 乏力、嗜睡
- 严重者抽搐、昏迷

### 2.3 潜伏期

| 类型 | 潜伏期 | 常见原因 |
|------|--------|----------|
| 细菌性 | 6-72小时 | 沙门氏菌、大肠杆菌 |
| 化学性 | 数分钟-数小时 | 农药、亚硝酸盐 |
| 真菌性 | 数小时-数天 | 毒蘑菇、霉变食物 |

## 三、食物中毒预防措施

### 3.1 采购安全

- 选择正规渠道购买
- 查看生产日期和保质期
- 注意食品外观和气味
- 索要购物凭证

### 3.2 储存安全

- 生熟食品分开
- 冷藏冷冻温度达标
- 定期清理过期食品
- 使用密封容器

### 3.3 加工安全

- 彻底加热食品（中心温度70°C以上）
- 生熟分开处理
- 保持厨房清洁
- 使用干净的水和食材

### 3.4 外出就餐

- 选择卫生条件好的餐厅
- 避免食用生冷食品
- 注意餐具卫生
- 保留消费凭证

## 四、食物中毒应急处理

### 4.1 立即措施

1. **停止食用**可疑食品
2. **保留样本**以便检测
3. **催吐**（清醒状态下，中毒2小时内）
4. **就医**并告知医生食用了什么

### 4.2 催吐方法

- 饮用大量温水
- 用手指刺激咽喉部
- 反复进行直到呕吐物变清

### 4.3 就医指征

- 症状严重或持续加重
- 出现脱水症状（口干、尿少、头晕）
- 高热不退
- 儿童、孕妇、老年人中毒

## 五、特殊食品中毒预防

### 5.1 毒蘑菇

- 不采摘、不食用野生蘑菇
- 购买正规渠道的食用菌
- 不要凭经验判断蘑菇是否有毒

### 5.2 发芽土豆

- 去除芽眼及周围部分
- 充分加热烹调
- 发芽严重应丢弃

### 5.3 四季豆

- 必须彻底煮熟
- 焖煮时间不少于10分钟
- 避免凉拌或爆炒

### 5.4 河豚鱼

- 禁止自行加工
- 购买正规渠道产品
- 由专业人员处理
