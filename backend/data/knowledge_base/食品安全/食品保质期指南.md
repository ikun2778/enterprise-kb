# 食品保质期指南

## 一、什么是食品保质期

食品保质期是指在标签指明的贮存条件下，保持品质的期限。在此期限内，产品完全适于销售，并保持标签中不必说明或已经说明的特有品质。

### 1.1 保质期与保存期的区别

- **保质期**：最佳食用期，超过此期限食品仍可能可以食用，但品质可能下降
- **保存期**：推荐的最终食用日期，超过此日期食品可能不再安全

### 1.2 常见食品保质期

| 食品类别 | 保质期 | 保存条件 |
|----------|--------|----------|
| 鲜牛奶 | 7-15天 | 2-6°C冷藏 |
| 酸奶 | 14-21天 | 2-6°C冷藏 |
| 面包 | 3-7天 | 常温密封 |
| 饼干 | 6-12个月 | 常温干燥 |
| 罐头 | 1-2年 | 常温避光 |
| 冷冻食品 | 3-12个月 | -18°C以下 |

## 二、如何判断食品是否变质

### 2.1 感官判断方法

1. **看**：观察颜色是否正常，有无霉斑、膨胀
2. **闻**：是否有异味、酸味、腐败味
3. **摸**：质地是否正常，有无粘液、软化
4. **尝**：少量尝试，是否有异常味道

### 2.2 常见变质特征

- **肉类**：表面发粘、颜色变暗、有异味
- **乳制品**：酸味加重、结块、分层
- **面包**：出现霉斑、变硬、有异味
- **罐头**：罐体膨胀、漏气、内容物变色

## 三、食品保存技巧

### 3.1 冷藏保存

- 生熟食品分开存放
- 使用密封容器
- 定期清理冰箱
- 注意温度设置（2-6°C）

### 3.2 冷冻保存

- 分装保存，避免反复解冻
- 标注日期，先进先出
- 保持-18°C以下温度

### 3.3 常温保存

- 避光、干燥、通风
- 远离热源
- 密封保存

## 四、特殊人群注意事项

### 4.1 孕妇

- 避免食用过期食品
- 注意生熟分开
- 避免食用高风险食品（生鱼片、半熟蛋等）

### 4.2 儿童

- 注意食品新鲜度
- 避免食用过多加工食品
- 注意过敏原

### 4.3 老年人

- 选择易消化食品
- 注意食品软硬度
- 避免食用过硬、过冷食品
