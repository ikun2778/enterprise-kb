# 劳动合同法常识

## 一、劳动合同的签订

### 1.1 签订时间

- 用人单位自用工之日起**一个月内**必须签订书面劳动合同
- 超过一个月不满一年未签订的，应当支付**双倍工资**
- 超过一年未签订的，视为已订立**无固定期限劳动合同**

### 1.2 劳动合同必备条款

1. 用人单位名称、住所和法定代表人
2. 劳动者姓名、住址和身份证号码
3. 劳动合同期限
4. 工作内容和工作地点
5. 工作时间和休息休假
6. 劳动报酬
7. 社会保险
8. 劳动保护、劳动条件和职业危害防护

### 1.3 试用期规定

| 合同期限 | 试用期上限 |
|----------|------------|
| 3个月-1年 | 1个月 |
| 1年-3年 | 2个月 |
| 3年以上/无固定 | 6个月 |

**注意事项**：
- 试用期工资不得低于约定工资的80%
- 同一用人单位只能约定一次试用期
- 试用期包含在劳动合同期限内

## 二、工资与加班

### 2.1 工资支付

- 工资应当以货币形式按月支付
- 不得克扣或无故拖欠
- 最低工资标准由各地政府规定

### 2.2 加班工资

| 加班类型 | 工资标准 |
|----------|----------|
| 工作日加班 | 1.5倍工资 |
| 休息日加班 | 2倍工资（或补休） |
| 法定节假日 | 3倍工资 |

### 2.3 法定节假日

全年法定节假日共**11天**：
- 元旦：1天
- 春节：3天
- 清明节：1天
- 劳动节：1天
- 端午节：1天
- 中秋节：1天
- 国庆节：3天

## 三、休假制度

### 3.1 带薪年休假

| 工龄 | 年休假天数 |
|------|------------|
| 1-10年 | 5天 |
| 10-20年 | 10天 |
| 20年以上 | 15天 |

**注意事项**：
- 国家法定节假日不计入年休假
- 未休年假应支付300%工资
- 年假可以跨年度安排

### 3.2 其他假期

- **婚假**：3天（各地可能有延长）
- **产假**：98天+各地延长
- **丧假**：1-3天
- **病假**：根据工龄享受医疗期

## 四、劳动合同解除

### 4.1 劳动者解除

- **提前30天**书面通知用人单位
- 试用期内提前**3天**通知
- 用人单位违法可立即解除

### 4.2 用人单位解除

**可以解除的情形**：
- 劳动者严重违反规章制度
- 劳动者严重失职
- 劳动者被依法追究刑事责任

**不得解除的情形**：
- 患职业病或工伤丧失劳动能力
- 女职工孕期、产期、哺乳期
- 医疗期内

### 4.3 经济补偿

**应当支付经济补偿的情形**：
- 用人单位提出解除
- 用人单位经济性裁员
- 劳动合同到期不续签（用人单位降低条件）

**补偿标准**：
- 每工作满一年支付一个月工资
- 六个月以上不满一年按一年计算
- 不满六个月支付半个月工资

## 五、社会保险

### 5.1 五险一金

| 险种 | 个人缴纳 | 单位缴纳 |
|------|----------|----------|
| 养老保险 | 8% | 16% |
| 医疗保险 | 2% | 8% |
| 失业保险 | 0.5% | 0.5% |
| 工伤保险 | 0 | 0.2-1.9% |
| 生育保险 | 0 | 0.8% |
| 住房公积金 | 5-12% | 5-12% |

### 5.2 社保权益

- 养老保险：累计缴满15年，退休后领取养老金
- 医疗保险：享受医疗报销
- 失业保险：非自愿失业可领取失业金
- 工伤保险：工伤享受医疗和补偿
- 生育保险：生育享受医疗和津贴

## 六、劳动争议处理

### 6.1 处理途径

1. **协商**：与用人单位直接协商
2. **调解**：向劳动争议调解委员会申请
3. **仲裁**：向劳动争议仲裁委员会申请
4. **诉讼**：对仲裁结果不服可向法院起诉

### 6.2 仲裁时效

- 劳动争议仲裁时效为**一年**
- 从当事人知道或应当知道权利被侵害之日起计算
- 拖欠工资争议不受一年时效限制

### 6.3 举证责任

- 用人单位掌握的证据由用人单位提供
- 工资支付记录由用人单位举证
- 考勤记录由用人单位举证
