# 国内旅行安全指南

## 一、出行前准备

### 1.1 证件准备

- 身份证（必备）
- 驾驶证（自驾游）
- 学生证/老年证（优惠票）
- 健康码/行程码（根据政策）
- 保险单据

### 1.2 物品准备

**必备物品**
- 手机及充电器
- 充电宝（不超过100Wh）
- 常用药品
- 适量现金
- 雨具

**衣物准备**
- 根据目的地天气准备
- 舒适的鞋子
- 防晒用品

### 1.3 信息准备

- 了解目的地天气
- 查询交通路线
- 预订住宿
- 了解当地风俗
- 记录紧急联系电话

## 二、交通安全

### 2.1 飞机出行

**禁止携带物品**
- 打火机、火柴
- 刀具、剪刀
- 超过100ml液体
- 充电宝超过200Wh

**注意事项**
- 提前2小时到达机场
- 贵重物品随身携带
- 保管好登机牌
- 注意航班动态

### 2.2 火车出行

**禁止携带物品**
- 易燃易爆物品
- 管制刀具
- 超过120ml的自喷压力容器
- 活物（导盲犬除外）

**注意事项**
- 提前1小时到达车站
- 保管好车票和身份证
- 注意行李安全
- 不要食用陌生人食物

### 2.3 自驾出行

**出发前检查**
- 轮胎气压和磨损
- 刹车系统
- 灯光系统
- 油液水平
- 备胎和工具

**行车安全**
- 系好安全带
- 不疲劳驾驶
- 不酒后驾驶
- 遵守交通规则
- 保持安全车距

## 三、住宿安全

### 3.1 选择住宿

- 选择正规酒店/民宿
- 查看评价和评分
- 确认安全设施
- 了解退订政策

### 3.2 入住检查

- 检查门锁是否正常
- 了解逃生通道
- 检查消防设施
- 贵重物品存放保险箱

### 3.3 住宿安全

- 不给陌生人开门
- 使用门链/门锁
- 注意个人隐私
- 保持房间整洁

## 四、景区安全

### 4.1 游玩安全

- 遵守景区规定
- 不去未开放区域
- 注意警示标志
- 不攀爬危险区域

### 4.2 水上安全

- 穿戴救生设备
- 不去深水区
- 不单独游泳
- 注意潮汐时间

### 4.3 登山安全

- 穿着合适的鞋子
- 携带足够饮水
- 不走野路
- 注意天气变化

## 五、紧急情况处理

### 5.1 迷路处理

- 保持冷静
- 使用手机导航
- 询问当地人
- 拨打110求助

### 5.2 财物丢失

- 立即报警
- 挂失银行卡
- 联系保险公司
- 补办证件

### 5.3 人身安全

- 避免冲突
- 寻求帮助
- 拨打110报警
- 保留证据

### 5.4 紧急电话

| 类型 | 电话 |
|------|------|
| 报警 | 110 |
| 急救 | 120 |
| 火警 | 119 |
| 交通事故 | 122 |
| 旅游投诉 | 12301 |
