# 家庭用电安全

## 一、基本用电常识

### 1.1 安全电压

- **安全电压**：36V以下
- **人体安全电流**：10mA以下
- **致命电流**：30mA以上

### 1.2 家庭用电标准

- 照明电路：220V
- 大功率电器：单独回路
- 总开关：带漏电保护

## 二、常见用电隐患

### 2.1 电线老化

**识别方法**：
- 外皮开裂、发硬
- 颜色变暗
- 使用超过15年

**处理措施**：
- 及时更换
- 请专业电工操作
- 使用国标电线

### 2.2 超负荷用电

**危险信号**：
- 经常跳闸
- 电线发热
- 插座变色

**预防措施**：
- 大功率电器单独回路
- 不使用多个大功率电器同一插座
- 计算总功率不超负荷

### 2.3 插座问题

**常见问题**：
- 插座松动
- 接触不良
- 过载使用

**安全标准**：
- 使用新国标插座
- 有安全门设计
- 不超负荷使用

## 三、电器使用安全

### 3.1 厨房电器

**电饭煲**：
- 不空烧
- 内胆干燥后再使用
- 远离水源

**微波炉**：
- 不使用金属容器
- 不加热密封食品
- 不空转

**电磁炉**：
- 使用配套锅具
- 不干烧
- 远离水源

### 3.2 卫生间电器

**电热水器**：
- 定期检查镁棒
- 使用前检查漏电保护
- 不长时间通电

**吹风机**：
- 不在潮湿环境使用
- 不堵塞进风口
- 使用后冷却再收纳

### 3.3 空调

**使用安全**：
- 定期清洗滤网
- 不长时间低温运行
- 定期检查线路

**节能建议**：
- 夏季设置26°C
- 配合风扇使用
- 定期维护保养

## 四、漏电保护

### 4.1 漏电保护器

**作用**：
- 检测漏电电流
- 自动切断电源
- 防止触电事故

**测试方法**：
- 每月按一次测试按钮
- 应立即跳闸
- 不跳闸需更换

### 4.2 接地保护

**重要性**：
- 将漏电电流导入大地
- 降低触电风险
- 保护电器设备

**检查方法**：
- 使用试电笔
- 检查三孔插座接地线
- 请专业人员检测

## 五、触电急救

### 5.1 脱离电源

**低压触电**：
- 关闭电源开关
- 用干燥绝缘物挑开电线
- 不直接接触触电者

**高压触电**：
- 通知供电部门
- 保持安全距离
- 不擅自处理

### 5.2 现场急救

1. 检查呼吸和意识
2. 无呼吸立即心肺复苏
3. 拨打120急救电话
4. 保持患者平卧
5. 注意保暖

### 5.3 心肺复苏步骤

1. **判断意识**：拍打肩部，呼叫
2. **呼叫帮助**：拨打120
3. **胸外按压**：
   - 位置：两乳头连线中点
   - 深度：5-6厘米
   - 频率：100-120次/分钟
4. **人工呼吸**：
   - 开放气道
   - 捏鼻吹气
   - 每30次按压后2次吹气

## 六、儿童用电安全

### 6.1 预防措施

- 使用安全插座保护盖
- 电线隐藏或固定
- 教育儿童不触摸插座
- 不在儿童面前拔插头

### 6.2 安全教育

- 告知电的危险性
- 教导正确使用电器
- 不在电器附近玩水
- 发现危险立即告知大人

## 七、雷雨天气用电

### 7.1 防雷措施

- 关闭门窗
- 拔掉电器插头
- 不使用太阳能热水器
- 不触摸金属管道

### 7.2 雷击急救

- 不接触伤者（可能带电）
- 立即拨打120
- 进行心肺复苏
- 处理烧伤
