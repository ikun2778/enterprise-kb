# 常见疾病预防

## 一、感冒预防

### 1.1 普通感冒

**症状**：
- 鼻塞、流涕
- 咽喉痛
- 轻微发热
- 咳嗽

**预防措施**：
- 勤洗手（20秒以上）
- 避免触摸面部
- 保持室内通风
- 适当锻炼
- 充足睡眠
- 均衡饮食

### 1.2 流行性感冒

**症状**：
- 突然高热（38.5°C以上）
- 全身酸痛
- 极度疲劳
- 头痛

**预防措施**：
- 每年接种流感疫苗
- 避免人群聚集
- 佩戴口罩
- 咳嗽礼仪

## 二、肠胃疾病预防

### 2.1 急性胃肠炎

**症状**：
- 恶心、呕吐
- 腹痛、腹泻
- 发热

**预防措施**：
- 注意饮食卫生
- 饭前便后洗手
- 不吃生冷食物
- 食物彻底加热

### 2.2 胃溃疡

**症状**：
- 上腹部疼痛
- 反酸、嗳气
- 腹胀

**预防措施**：
- 规律饮食
- 避免刺激性食物
- 戒烟限酒
- 减少压力

## 三、心血管疾病预防

### 3.1 高血压

**危险因素**：
- 高盐饮食
- 肥胖
- 缺乏运动
- 吸烟饮酒
- 精神紧张

**预防措施**：
- 每日盐摄入<6克
- 控制体重
- 适量运动
- 戒烟限酒
- 定期测量血压

### 3.2 冠心病

**症状**：
- 胸闷、胸痛
- 心悸
- 气短

**预防措施**：
- 控制血压、血糖、血脂
- 健康饮食
- 规律运动
- 控制体重
- 定期体检

## 四、糖尿病预防

### 4.1 糖尿病类型

- **1型**：自身免疫性疾病，多发于青少年
- **2型**：与生活方式相关，多发于成年人
- **妊娠期**：孕期发生，产后可能恢复

### 4.2 高危人群

- 有家族史
- 肥胖
- 40岁以上
- 有妊娠糖尿病史
- 高血压、高血脂

### 4.3 预防措施

- 控制体重（BMI<24）
- 健康饮食
- 每周运动150分钟
- 定期检查血糖
- 避免久坐

## 五、癌症预防

### 5.1 一级预防

- 戒烟限酒
- 健康饮食
- 适量运动
- 控制体重
- 接种疫苗（如HPV疫苗）

### 5.2 二级预防

**定期筛查项目**：
- 肺癌：低剂量CT（高危人群）
- 乳腺癌：乳腺X线摄影
- 宫颈癌：HPV检测+TCT
- 结直肠癌：肠镜检查
- 胃癌：胃镜检查
- 肝癌：超声+甲胎蛋白

### 5.3 癭症预警信号

1. 身体出现异常肿块
2. 不明原因出血
3. 持续性消化不良
4. 持续性咳嗽或声音嘶哑
5. 不明原因体重下降
6. 大便习惯改变
7. 痣发生变化

## 六、心理健康

### 6.1 压力管理

- 规律作息
- 适当运动
- 培养爱好
- 社交活动
- 冥想放松

### 6.2 焦虑抑郁识别

**焦虑症状**：
- 过度担忧
- 紧张不安
- 失眠
- 注意力不集中

**抑郁症状**：
- 情绪低落
- 兴趣减退
- 疲劳无力
- 自我否定

### 6.3 求助途径

- 心理咨询热线：12320-5
- 医院心理科
- 心理咨询机构
- 亲友支持

## 七、健康体检建议

### 7.1 基础体检项目

| 年龄 | 建议频率 | 重点项目 |
|------|----------|----------|
| 20-30岁 | 每2年 | 血常规、尿常规、肝功能 |
| 30-40岁 | 每年 | 增加血糖、血脂、心电图 |
| 40-50岁 | 每年 | 增加肿瘤标志物、眼底检查 |
| 50岁以上 | 每年 | 增加骨密度、胃肠镜 |

### 7.2 体检注意事项

- 体检前8小时禁食
- 避免剧烈运动
- 告知医生病史和用药
- 保留体检报告对比
