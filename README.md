# 企业智能知识库助手

基于 RAG + Agent 的企业级知识库问答系统

## 项目简介

这是一个基于检索增强生成（RAG）和Agent技术的知识库问答系统。系统支持智能检索与问答，集成了查询路由、查询重写、多种回答模式等能力。

## 知识库内容

当前知识库包含**生活百科常识**，涵盖：

| 类别 | 内容 | 文档数 |
|------|------|--------|
| 食品安全 | 食品保质期、食物中毒预防 | 2篇 |
| 出行指南 | 国内旅行安全指南 | 1篇 |
| 法律常识 | 劳动合同法常识 | 1篇 |
| 健康生活 | 常见疾病预防 | 1篇 |
| 家居安全 | 家庭用电安全 | 1篇 |
| 消费维权 | 消费者权益保护法 | 1篇 |

## 技术架构

```
用户提问
    ↓
查询路由 (list/detail/general)
    ↓
查询重写 (模糊查询优化)
    ↓
混合检索 (向量+BM25+RRF)
    ↓
父子文档 (小块检索→大块上下文)
    ↓
根据路由类型选择回答方式:
  - list → 列表回答 (不调LLM)
  - detail → 分步回答
  - general → 基础回答
    ↓
返回回答 + 来源文档
```

## 核心功能

- **父子文档**：小块检索，大块上下文
- **查询路由**：自动识别查询类型
- **查询重写**：优化模糊查询
- **混合检索**：向量检索 + BM25 + RRF重排序
- **多种回答**：列表/分步/基础三种模式
- **流式输出**：实时显示回答
- **多轮对话**：支持上下文记忆

## 技术栈

| 组件 | 技术选型 |
|------|----------|
| Agent框架 | LangChain |
| Embedding | HuggingFace BGE |
| 向量数据库 | FAISS |
| 稀疏检索 | BM25 |
| LLM | Moonshot (Kimi) |

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/ikun2778/enterprise-kb.git
cd enterprise-kb/backend
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置API Key

```bash
cp .env.example .env
# 编辑 .env 文件，填入 MOONSHOT_API_KEY
```

### 4. 启动终端问答

```bash
python cli.py
```

## 命令说明

| 命令 | 说明 |
|------|------|
| 直接输入问题 | 智能问答 |
| `new` / `新对话` | 开始新对话 |
| `stats` / `统计` | 查看知识库统计 |
| `stream` / `流式` | 切换流式输出模式 |
| `help` / `帮助` | 显示帮助信息 |
| `quit` / `退出` | 退出程序 |
