# 企业智能知识库助手

基于 RAG + Agent 的企业级知识库问答系统

## 项目简介

这是一个基于检索增强生成（RAG）和Agent技术的企业知识库问答系统。系统支持多格式文档的智能检索与问答，集成了工具调用、多轮对话等Agent能力。

## 核心功能

- **智能问答**：基于知识库的精准回答
- **多轮对话**：支持上下文理解和追问
- **工具调用**：集成搜索、计算、时间查询等工具
- **混合检索**：向量检索 + BM25 + RRF重排序
- **来源引用**：回答附带文档来源

## 技术栈

| 组件 | 技术选型 |
|------|----------|
| Agent框架 | LangChain |
| Embedding | HuggingFace BGE |
| 向量数据库 | FAISS |
| 稀疏检索 | BM25 |
| LLM | Moonshot (Kimi) |

## 快速开始

### 1. 环境准备

```bash
cd backend
cp .env.example .env
# 编辑 .env 文件，填入你的 MOONSHOT_API_KEY
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动终端问答

```bash
python cli.py
```

### 4. 使用示例

```
❓ 您的问题: 什么是RAG？

⏳ 思考中...

💬 回答:
RAG（Retrieval-Augmented Generation）是一种将检索与生成相结合的技术...

📎 来源文档:
  [1] chapter1 (相关度: 85.23%)
  [2] chapter4 (相关度: 72.15%)

🔧 使用工具: 知识库搜索
```
